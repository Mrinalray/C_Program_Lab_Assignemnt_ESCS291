# 🧾 C Program: Hello World

## 📖 Description
This program prints "Hello, World!" on the screen using the C programming language.

---

## 💻 Code
```c
#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}
```
## ⚙️ How it Works
- `#include <stdio.h>` → used for input/output functions  
- `int main()` → starting point  
- `printf()` → prints text  
- `return 0;` → ends program
- 
## ▶️ Output
Hello, World
