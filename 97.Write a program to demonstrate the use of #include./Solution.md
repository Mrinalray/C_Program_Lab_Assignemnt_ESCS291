## 📌 Program 97: Demonstrate the Use of `#include`

### 💻 Code

```c id="i4m8qx"
#include <stdio.h>   // Standard input-output header

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

### 📖 Explanation

* `#include <stdio.h>` → includes standard library for input/output
* Allows use of functions like `printf()` and `scanf()`
* `#include` is a preprocessor directive
* It tells the compiler to include the contents of a file before compilation

### ▶️ Output

```id="m3q2zn"
Hello, World!
```
